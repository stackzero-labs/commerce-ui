---
title: Face Rating
description: A component that allows you to display a rating.
---

The component uses `lucide-react` icons to display the rating's faces. You can of course use any other icon library or custom icons.

## Usage


```tsx
import { FaceRating } from "@/components/ui/face-rating"
```

```tsx
<FaceRating defaultRating={4} maxRating={5} iconSize={24} />
```

## Examples

### Custom colors

You can override any of the colors used in the rating component. 
Just pass the prop `colors` with the color(s) you want to use.

<ComponentPreview
  name="face-rating-custom-colors"

### Base color

By passing the prop `baseColor` (e.g., your brand's color), the component will use the base color to calculate the other colors.
